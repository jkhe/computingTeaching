1 answers = ["Definitely yes", "No chance", "Ask again later"]

7 input.on_gesture(Gesture.SHAKE, on_gesture_shake)
That line is OK — look again.

Found it!

The startup message belongs at the very top of the file, not inside the shake function. Inside the function it would run on every shake, not just at startup. The correct program starts with basic.show_string("My 8-Ball") as the very first line.
